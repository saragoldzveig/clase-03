Gráfica figurativa

Sara Goldzveig

Usando Guemil para una doble representación, podemos generar un enfoque para la visualización de datos. Guemil permite combinar diferentes elementos visuales, como gráficos SVG y representaciones icónicas, para transmitir información de manera efectiva. Esta capacidad de crear una doble representación nos ofrece una perspectiva más completa de los datos, permitiéndonos comunicar conceptos complejos de manera concisa.
